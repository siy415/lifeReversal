## 데분프 2팀 인생력전 프로젝트 커밋용 git
- 심인용, 장동찬, 장병훈, 정보경


## 기본 사항
- 프로젝트를 진행하면서 발생한 문서는 'docs' 폴더에 넣어주세요
- 필요한 아이콘 혹은 ui자료들은 'ico' 폴더 혹은 해당 폴더 안에 서브 폴더를 만들어서 저장 부탁드립니다.
- 코드는 'code'폴더 안에 기능별 folder tree를 생성하여 저장할 예정입니다.


## 추가 사항
- 추가적인 내용이 있을 경우 해당 파일 수정 부탁드립니다.
